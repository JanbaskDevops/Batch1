1) Build context: It's a text file with name Dockerfile(Dockerfile can also work)

    ubuntu $ ls
    Dockerfile  dockerfile  

    ubuntu $ cat Dockerfile
    FROM ubuntu
    CMD echo "Ubuntu Dockerfile"

    ubuntu $ cat dockerfile
    FROM ubuntu
    CMD echo "Ubuntu dockerfile"

    Output: Ubuntu Dockerfile

2) FROM statement; CMD <command>


2) Write docker file to prove OS level virtualization.

3) Syntax for build context
      FROM OS
      RUN Provision
      CMD application

Command to run docker image >>> docker run <ImageName>:<Tag>
