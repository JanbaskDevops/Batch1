Here's a step-by-step guide to installing JFrog Artifactory on Ubuntu:

1. *Update System Packages:*
   bash
   sudo apt-get update
   

2. *Install Java:*
   JFrog Artifactory requires Java. You can install OpenJDK using the following command:
   bash
   sudo apt-get install openjdk-11-jdk -y
   

3. *Download JFrog Artifactory:*
   Go to the [JFrog Artifactory download page](https://jfrog.com/artifactory/download/) and get the download link for the latest version. Use wget to download it:
   bash
   wget https://releases.jfrog.io/artifactory/artifactory-pro/org/artifactory/pro/jfrog-artifactory-pro/7.21.10/jfrog-artifactory-pro-7.21.10-linux.tar.gz
   

4. *Extract the Downloaded Archive:*
   bash
   tar -xvzf jfrog-artifactory-pro-7.21.10-linux.tar.gz
   

5. *Move the Extracted Folder:*
   bash
   sudo mv artifactory-pro-7.21.10 /opt/jfrog
   

6. *Set Up Permissions:*
   bash
   sudo chown -R $(whoami):$(whoami) /opt/jfrog
   

7. *Start Artifactory:*
   ```bash
   /opt/jfrog/artifactory-pro-7.21.10/app/bin/artifactory.sh start
